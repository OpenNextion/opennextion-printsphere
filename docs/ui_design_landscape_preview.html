PrintSphere ONX 480x320 横屏 UI 静态设计稿

草案：用于确认 ONX3248G035 横屏方向下的 PrintSphere 页面重排。设计基于同一套原项目页面和数据接口，不引入新的打印机控制协议。

说明：所有屏幕 mock 固定为 480 x 320。蓝色虚线表示 44px 以上主要触控区域；正式 LVGL 不显示虚线。图标和字体落地时优先复用 `bambuicon_small`、Dosis、MDI 字体子集。
